ğŸ› TROUBLESHOOTING:
================================================================================

Problem: Backend won't start
----------------------------
Solution:
1. Check Python version: python --version (should be 3.8+)
2. Check MongoDB is running:
   - MongoDB service should be active
   - Try connecting: mongo (or mongosh for newer versions)
3. Check port 8000 is not in use
4. Reinstall dependencies: pip install -r requirements.txt


Problem: "Import could not be resolved" errors
----------------------------------------------
Solution:
  These are linter warnings, not errors. Backend will work fine.
  Dependencies will be installed when you run: pip install -r requirements.txt


Problem: Frontend can't connect to backend
------------------------------------------
Solution:
1. Ensure backend is running (check http://localhost:8000)
2. Check browser console for errors (F12)
3. Verify api-config.js BASE_URL is correct: http://localhost:8000
4. Check CORS settings in backend (should allow localhost)


Problem: MongoDB connection failed
----------------------------------
Solution:
1. Ensure MongoDB service is running
2. Check connection string in .env file
3. For MongoDB Atlas:
   - Whitelist your IP address
   - Use correct connection string format
   - Check database user credentials


Problem: File upload not working
--------------------------------
Solution:
1. Check uploads directory exists in backend folder
2. Verify file size limits in .env
3. Check browser console for specific error messages
